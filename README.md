# Мобильное приложение-аудиоплеер

Учебный проект по дисциплине «Системная и программная инженерия».

## Описание проекта

Проект представляет собой мобильное приложение-аудиоплеер для прослушивания музыки, подкастов и аудиокниг. Приложение должно предоставлять пользователю поиск контента, управление воспроизведением, плейлисты, избранное, историю прослушиваний, таймер сна, фоновое воспроизведение и синхронизацию данных между устройствами.

## Выбранная методология

Для управления разработкой выбрана методология Scrum. Она подходит для небольшой команды, так как позволяет вести разработку короткими итерациями, распределять задачи между участниками, регулярно проверять результат и быстро изменять план при уточнении требований.

## Основной стек технологий

- Mobile: Android, Kotlin, Jetpack Compose
- Backend: Java, Spring Boot
- Database: PostgreSQL
- Cache: Redis
- File Storage: MinIO / S3
- Search: Elasticsearch
- DevOps: Docker, Docker Compose, GitHub Actions
- Documentation: Doxygen
- Monitoring: Prometheus, Grafana, ELK

## Основные функции приложения

- регистрация и авторизация пользователя;
- поиск музыки, подкастов и аудиокниг;
- управление воспроизведением;
- создание плейлистов;
- добавление контента в избранное;
- история прослушиваний;
- сохранение позиции для подкастов и аудиокниг;
- таймер сна;
- фоновое воспроизведение;
- синхронизация данных между устройствами.
